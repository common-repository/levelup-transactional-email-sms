Providing WordPress users the ability to utilize Levelup's transactional Email & SMS services. 
 
== Description ==

= Connect to Levelup's services from your WordPress site =

This plugin is created to connect your WordPress site to the power of Levelup's services.

You will be able to utilize Levelup's network to send email's and SMS'es with high delivery rates.

= Instant email configuration =

Once you have created your Levelup account and entered your API credentials. Email just work! 

We hook into the core `wp_mail` functionality, so that all emails automatically gets sent through our network.

= Gain the power to send SMS = 

With this plugin you also gain the awesome power of the `wp_sms` function. 

This function lets you quickly send SMS from your PHP code.

= Logs from the Levelup Console =

View logs with information about emails and sms'es sent from the [Levelup Console](https://console.levelup.com "Your Levelup Console")!
 
== Installation ==
 
This section describes how to install the plugin and get it working.
 
e.g.
 
1. Upload the plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Enter your credentials under the `Levelup` menu in WordPress
 
== Frequently Asked Questions ==
 
= How do I acquire my credentials =
 
Create a new user account at the [Levelup Console](https://console.levelup.no "Your Levelup Console")

Create a new account for your person or business.

Provide payment information.

Create a workspace.

Request access to the Email and SMS products.
 
= Is there a free version? =
 
It's free to install the plugin itself, but you have to pay for the services used in the [Levelup Console](https://console.levelup.no "Your Levelup Console").
